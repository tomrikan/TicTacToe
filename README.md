# Ristinolla
Ristinolla on kahden pelaajan peli, jossa pelaajat sijoittavat pelilaudalle vuorotellen, joko ristin tai ympyrän. Tavoitteena on saada kolme omaa merkkiä samalle pysty-, vaaka- tai vinoriville.
